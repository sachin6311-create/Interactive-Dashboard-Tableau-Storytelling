# Interactive-Dashboard-Tableau-Storytelling

Q1. Difference between Tableau Dashboard and Story Point

A Tableau Dashboard is a single screen that combines multiple visualizations to provide an
interactive view of data. It is used for monitoring and analysis. A Story Point is a sequence of
dashboards or sheets arranged in a logical flow to tell a data story. In healthcare, a dashboard
could show patient counts and costs, while a story could explain trends like rising treatment costs
over time.

Q2. Actions in Tableau and Filter Action

Actions in Tableau are interactive features that allow users to interact with visualizations. A Filter
Action allows users to click on one chart and filter data in another chart. In hospital data, clicking on
a department can filter patient records for that department only.

Q3. Importance of Distribution Chart

A distribution chart helps understand how data is spread across categories. In healthcare, it shows
how patients are distributed across regions, helping identify high-demand areas.
Q4. Department Performance Visualization
This visualization helps compare treatment cost and recovery rates across departments using
colors. It helps management identify which departments are performing well and which need
improvement.

Q5. Treatment Trend Over Time

A line chart shows trends in treatment costs over time. Filtering data from March to September
helps analyze seasonal patterns and cost fluctuations.
Q6. Outcome Analysis

A pie chart shows distribution of patient outcomes such as recovered, critical, and under treatment.
This helps evaluate hospital performance and patient care quality.
Q7. Doctor Performance

Ranking doctors by treatments handled helps identify top performers. A bar chart makes
comparison easy and supports performance evaluation.

Q8. Interactive Dashboard

An interactive dashboard combines multiple charts with filters like hospital region and department. It
allows users to explore data dynamically and gain insights quickly.

Q9. Storytelling Flow

A Tableau story presents insights in a sequence: patient distribution, department analysis, outcome
analysis, and doctor performance. It helps stakeholders understand the full picture step-by-step.
Q10. Business Questions Answers

1. Department generating highest cost: Depends on data but usually high-specialty departments
like ICU. 2. Region with most critical patients: Identified using outcome distribution chart. 3.
Business decisions: Improve underperforming departments, allocate resources to high-demand
regions, and optimize treatment costs.
